- [Java JDK 11+](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html)

### Installation

```bash
# Clone the repository:
git clone https://github.com/OmarAfifi-CSE/Digging-Jim.git

# Open the project in NetBeans:
```
